# Higney International — Website

Static mirror of www.higney-international.com (originally Squarespace), captured for rebuild.

## Pages
- `index.html` — Home
- `about.html` — About
- `projects.html` — Projects
- `fulfillment.html` — Fulfillment
- `team.html` — Team
- `contact.html` — Contact

All CSS, JS, and images are in `assets/`. Internal nav links are rewritten to the local `.html` files.

## Notes
This is a captured snapshot of the rendered Squarespace site. The Squarespace CMS, forms backend, and dynamic features won't function as-is. Use this as the content + design reference for a clean static rebuild.

`mirror.py` is the capture script used to generate this.
